Basic Skills

Essential techniques for working efficiently in CoSpaces. These fundamental skills will help you build and program your 3D scenes more effectively.

Attaching Objects to Faces

Video Demo: Double-click to select the attach option

How to Attach Objects to Specific Faces:
  1. Double-click on the object you want to attach to
  2. Select the "attach" option from the popup menu
  3. Click on the specific face of the object where you want to attach
  4. Now you can place new objects precisely on that face
When to Use:
  • Building complex structures with precise placement
  • Creating buildings with objects on walls
  • Placing items on surfaces like tables or shelves

Making Objects Codeable

Video Demo: Double-click to make objects codeable

How to Make Objects Codeable:
  1. Double-click on any object in your scene
  2. Look for the "Code" option in the popup menu
  3. Click 'Use in CoBlocks' to make the object available in code
  4. The object will now appear in your coding panel for programming
When to Use:
  • When you need to code an object

Infinite Animation

Make a character play an animation that repeats forever, like walking, dancing, or waving. This is great for background characters that need to stay active.

Block Setup

Screenshot: Infinite Animation Blocks

Key Blocks:
  • When scene starts
  • Play animation (walk/dance/wave)

EXTENSION

Screenshot: Example of Walking Character

Steps:
  1. Add "When scene starts" block
  2. Add "Forever" loop block
  3. Inside loop, add "Play animation" block
  4. Select animation type (e.g., "walking")
  5. Add "Wait" block so you only repeat a certain part of the animation

When to Use

  • For background characters that need continuous movement
  • To make your scene feel alive and active
  • For decorative elements like animals or people in a scene
  • For characters that need to show continuous behavior (like dancing)

Click to Talk

Make a character display a speech bubble or play a sound when clicked/tapped. This creates interactive characters that can provide information or dialogue.

Block Setup

Screenshot: Click to Talk Blocks

Key Blocks:
  • When item clicked
  • Say (speech bubble)
  • Play sound (optional)

EXTENSION

Screenshot: Character with Speech Bubble

Steps:
  1. Add "When item clicked" block to your character
  2. Add "Say" block and type your message
  3. Optional: Add "Play sound" for voice effect
  4. Add "Wait" block (e.g., 3 seconds) to keep message visible
  5. Optional: Add "Say nothing" to clear bubble after waiting

When to Use

  • For interactive characters in a digital story
  • To provide information about objects or places in your scene
  • To create dialogue between characters in a game
  • For tour guides in virtual field trips

Making a Quiz

Create an interactive quiz with questions, multiple-choice answers, and score tracking. This is perfect for educational games, assessments, or review activities.

Block Setup

Screenshot: Quiz System Blocks

Key Blocks:
  • Quiz block with the question, the two answers, and the correct answer selected
  • Say feedback (to tell the user if they got it right or wrong)

EXTENSION

Try this one yourself using the hints below!

Steps for Basic Quiz:
  1. Create a variable called "score" starting at 0
  2. Add a quiz
  3. Make sure to set the correct answer
  4. If correct, add +1 to the score variable and show positive feedback (play an animation or get the character to say "Correct!")
  5. If wrong, show feedback about the correct answer and take -1 from the score
  6. Get the character to say the score variable at the end
  7. EXTRA EXTENSION: Insert another quiz inside the quiz so there are two or more questions!

When to Use

  • For educational activities to test knowledge
  • In scavenger hunts or treasure hunts
  • To create interactive review games
  • For class assessments or learning checks

Character Paths

Make characters move along specific paths in your scene. You can create patrols, guided tours, or animated journeys that follow predefined routes.

Find the paths in the 'Special' tab in the library.

Double click the path and rename it to make it easier to use in code.

Search for 'path' in code and add this code to the thing you want to animate (not the path itself!)


When to Use

  • For background visuals to show activity and life in your scene
  • To create guided tours of virtual spaces
  • For animations showing historical journeys or processes
  • To demonstrate routes or paths in geography/map projects
