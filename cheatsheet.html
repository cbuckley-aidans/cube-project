Infinite Animation

Make a character play an animation that repeats forever, like walking, dancing, or waving. This is great for background characters that need to stay active.

Block Setup

Screenshot: Infinite Animation Blocks

Key Blocks:
  • When scene starts
  • Forever loop
  • Play animation (walk/dance/wave)
  • Wait (optional - for animation timing)

EXTENSION

Screenshot: Example of Walking Character

Steps:
  1. Add "When scene starts" block
  2. Add "Forever" loop block
  3. Inside loop, add "Play animation" block
  4. Select animation type (e.g., "walking")
  5. Optional: Add "Wait" block if animation needs timing

When to Use

  • For background characters that need continuous movement
  • To make your scene feel alive and active
  • For decorative elements like animals or people in a scene
  • For characters that need to show continuous behavior (like dancing)

Click to Talk

Block Setup

Screenshot: Click to Talk Blocks

Key Blocks:
  • When item clicked
  • Say (speech bubble)
  • Play sound (optional)
  • Wait (to control how long message displays)

EXTENSION

Screenshot: Character with Speech Bubble

Steps:
  1. Add "When item clicked" block to your character
  2. Add "Say" block and type your message
  3. Optional: Add "Play sound" for voice effect
  4. Add "Wait" block (e.g., 3 seconds) to keep message visible
  5. Optional: Add "Say nothing" to clear bubble after waiting

When to Use

  • For interactive characters in a digital story
  • To provide information about objects or places in your scene
  • To create dialogue between characters in a game
  • For tour guides in virtual field trips

Making a Quiz

Create an interactive quiz with questions, multiple-choice answers, and score tracking. This is perfect for educational games, assessments, or review activities.

Block Setup

Screenshot: Quiz System Blocks

Key Blocks:
  • Quiz block with the question, the two answers, and the correct answer selected
  • Say feedback (to tell the user if they got it right or wrong)

EXTENSION

Screenshot: Simple Quiz Setup

Steps for Basic Quiz:
  1. Create a variable called "score" starting at 0
  2. Add question text using a 3D text object
  3. Add objects for each answer choice (A, B, C)
  4. For each answer object, add "When item clicked" block
  5. Inside, use "If/Then" to check if it's the correct answer
  6. If correct, add +1 to score and show positive feedback
  7. If wrong, show feedback about the correct answer
  8. Add "Wait" then code to show the next question

When to Use

  • For educational activities to test knowledge
  • In scavenger hunts or treasure hunts
  • To create interactive review games
  • For class assessments or learning checks

Character Paths

Make characters move along specific paths in your scene. You can create patrols, guided tours, or animated journeys that follow predefined routes.

Block Setup

Screenshot: Character Path Blocks

Key Blocks:
  • When scene starts (or other trigger)
  • Move to position
  • Wait blocks (for timing)
  • Turn to face direction (optional)
  • Play animation (optional)

EXTENSION

Screenshot: Character Following a Path

Steps for Basic Path:
  1. Add "When scene starts" block (or another trigger)
  2. Add "Play animation" block (e.g., "walking")
  3. Add "Move to position" block with coordinates of first waypoint
  4. Add "Wait" block to give character time to reach position
  5. Add "Turn to face" block to rotate toward next waypoint (optional)
  6. Repeat steps 3-5 for each waypoint in the path

Advanced Path Features

Screenshot: Loop Path with Talking

Advanced Features:
  • Add a Forever loop to make characters patrol
  • Use "Say" blocks at specific waypoints to narrate
  • Add "If/Then" blocks to create conditional paths
  • Use variables to track which waypoint the character is at

When to Use

  • For non-player characters (NPCs) in games
  • To create guided tours of virtual spaces
  • For animations showing historical journeys or processes
  • To demonstrate routes or paths in geography/map projects
