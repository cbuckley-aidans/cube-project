CHUNK 1: WHAT IS BRANCHING?

TEACHER CHUNK
5 mins

Branching: Making Decisions in Code

Branching is like coming to a fork in a road - your program needs to decide which way to go. Just as you might choose different paths when walking to school depending on the weather, in CoSpaces we can make our programs choose different actions based on different conditions.

1

Branching in Real Life

Think of branching as "if-then" decisions:

Weather Choice

IF it's raining THEN take an umbrella

Game Decision

IF score > 10 THEN level up

Quiz Answer

IF answer correct THEN award point

2

If Statements

In CoSpaces, we use "if statements" like this:

If statements need a condition that's either true or false!

3

Parts of an If Statement

An if statement has three important parts:

1

IF

Starts the decision-making process

2

CONDITION

What we're checking (must be true or false)

3

THEN

What happens if the condition is true

4

If-Else Statements

Sometimes we want to do one thing if a condition is true, and something else if it's false:

1

IF TRUE

This code runs if the condition is true

2

ELSE

This code runs if the condition is false

Why Branching Is Awesome

1

MAKES DECISIONS

Programs can decide what to do based on different situations

2

RESPONDS TO USERS

Programs can react differently to what players do

3

CREATES QUIZZES

Check if answers are correct and give feedback

4

CONTROLS GAMES

Check scores, lives, and other game values

Building A Multi-Question Quiz

Follow these steps to create a quiz with multiple questions:

  1. Plan Your Questions:
    • Think about a question that you want to ask if the user gets the first question RIGHT
    • Think about a question that you want to ask if the user gets the first question WRONG
  2. Think:
    • How could you arrange this quiz setup?
  3. Extension:
    • How could you make this even more complex?
    • Consider a variable that keeps track of the score
    • An if statement at the end that plays different animations based on the amount of questions you got right
