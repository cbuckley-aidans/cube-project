CHUNK 1: WHAT IS BRANCHING?

TEACHER CHUNK
5 mins

Branching: Making Decisions in Code

Branching is like coming to a fork in a road - your program needs to decide which way to go. Just as you might choose different paths when walking to school depending on the weather, in CoSpaces we can make our programs choose different actions based on different conditions.

1

Branching in Real Life

Think of branching as "if-then" decisions:

Weather Choice

IF it's raining THEN take an umbrella

Game Decision

IF score > 10 THEN level up

Quiz Answer

IF answer correct THEN award point

2

If Statements

In CoSpaces, we use "if statements" like this:

Screenshot: Simple if statement block that checks if score equals 10

If statements need a condition that's either true or false!

3

Parts of an If Statement

An if statement has three important parts:

Screenshot: Annotated if statement showing the three parts: IF, CONDITION, and THEN blocks

1

IF

Starts the decision-making process

2

CONDITION

What we're checking (must be true or false)

3

THEN

What happens if the condition is true

4

Conditions We Can Check

In CoSpaces, we can check many different conditions:

1

EQUALS

2

3

LESS THAN

Checks if one value is smaller than another

Screenshot: Less than condition block
