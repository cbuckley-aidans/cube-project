CHUNK 1: WHAT IS A VARIABLE?

TEACHER CHUNK
5 mins

Variables: Magic Containers for Information

A variable is like a magical box with a label that can hold different things. Just as you might have a box labelled "TOYS" for your toys, in CoSpaces we create variables with special names to remember important information in our games and stories.

1

Variable Imagination

Think of variables as special containers:

score

Keeps track of points (like 5)

lives

Keeps track of player lives (like 3)

playerName

Remembers player name (like "Sam")

2

Creating Variables

In CoSpaces, we create variables like this:

Search for "variable". You're looking for: set variable myVar to " ".

3

Types of Variables

Variables can hold different kinds of information:

1

NUMBERS

Numbers like 1, 10, or 42 that you can count with

2

TEXT

Words and letters like "Hello" or "Player"

3

YES/NO

True or False values (like a light switch - on or off)

4

COLOURS

Colours like red, blue or green to use in your world

Why Variables Are Awesome

1

REMEMBER THINGS

Variables let your program remember important information

2

KEEP SCORE

Variables can count points in games and keep track of scores

3

CHANGE THINGS

Variables can change while your program is running

4

MAKE DECISIONS

Variables help your program make choices and decisions

Making Variables Change

Key Points We Learned

  • Variables are like containers that store information
  • Variables can hold numbers, text, and other types of data
  • We can change variable values during our program
